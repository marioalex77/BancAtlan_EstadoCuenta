# PASOS PARA INSTALAR APLICATIVO DE ESTADO DE CUENTA BANCO ATLANTIDA

# Base de Datos
1 - Descargar docker desktop desde la web de docker hacia tu maquina e intalarlo.
2 - Descargar git desde la web de git hacia tu maquina e instalo
3 - Clona le repositorio de git en tu maquina:
	git clone https://github.com/marioalex77/BancAtlan_EstadoCuenta.git
4 - Abrir directorio donde se descargó el proyecto con tu consola de windows
5 - Ejecuta el archivos docker compose
	docker compose up -d
